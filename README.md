# C盘清理工具 (批处理版)

这是一个强大的Windows C盘清理工具，通过批处理脚本实现多种清理功能。

## 功能特点
- 临时文件清理
- 系统更新缓存清理
- 缩略图缓存清理
- 系统日志清理
- 深度清理（包括Windows.old）
- 自动重启功能
- 空间占用分析

## 使用方法
1. 下载 `CleanC.bat` 文件
2. 右键点击文件，选择"以管理员身份运行"
3. 根据菜单选择清理选项

```batch
右键 -> 以管理员身份运行
选择清理选项 -> 等待完成
```

## 安全提示
- 深度清理会删除系统还原点
- 清理前建议备份重要数据
- 建议在系统升级后使用Windows.old清理
