# NekoClean
Windows 清理工具 - PS脚本
这是一个强大的Windows清理工具，通过脚本实现多种清理功能。

## 功能特点
- 临时文件清理
- 系统更新缓存清理
- 缩略图缓存清理
- 系统日志清理
- 深度清理（包括Windows.old）
- 自动重启功能
- 空间占用分析

## 使用方法
1. 下载 `NekoClean.ps1` 文件
2. 右键点击文件，选择"使用 PowerShell 运行"
3. 根据菜单选择清理选项

```batch
右键 -> 使用 PowerShell 运行
选择清理选项 -> 等待完成
```

## 安全提示
- 深度清理会删除系统还原点
- 清理前建议备份重要数据
- 建议在系统升级后使用Windows.old清理
